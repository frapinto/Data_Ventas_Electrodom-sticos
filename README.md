# Análisis de Ventas y Marketing

Este proyecto tiene como objetivo realizar un análisis exploratorio y de rendimiento de datos de ventas, clientes y marketing para identificar tendencias clave y optimizar estrategias comerciales.

## Estructura del Proyecto:

*   **Etapa 1 - Análisis Exploratorio Inicial:** Se realiza una revisión de la calidad de los datos, identificando nulos y duplicados en los datasets de `ventas`, `clientes` y `marketing`.
*   **Etapa 2 - Limpieza y Transformación de Datos:** Se eliminan duplicados y valores nulos críticos, se estandarizan formatos de datos y se crean métricas clave como `importe` (precio * cantidad).
*   **Etapa 3 - Agregación e Integración de Datos:** Se combinan los datos de ventas y marketing para evaluar la efectividad de las campañas, identificando los productos de alto rendimiento y resumiendo ventas por categoría.
*   **Etapa 4 - Análisis y Visualización de Datos:** Se aplican estadísticas descriptivas, análisis exploratorio (EDA) y correlaciones. Los hallazgos se presentan mediante visualizaciones interactivas y estáticas, destacando el rendimiento de productos, la estacionalidad de ventas y el perfil de los clientes.

## Conclusiones Clave:

*   Identificación de productos estrella y aquellos con bajo retorno de inversión en marketing (ROAS).
*   Análisis de tendencias de ventas mensuales, revelando periodos de mayor actividad.
*   Perfil demográfico y geográfico de los clientes, indicando oportunidades de focalización.

Este análisis provee información valiosa para la toma de decisiones estratégicas en ventas y marketing.
